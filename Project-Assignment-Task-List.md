# Project Assignment Task List

Here is the verbatim text from the Assignment file:

```markdown
### 5.2.3 What you need to do

We do not ask you to re-train models. The reason for this is that this is a very computationally expensive operation, which you may not have the resources for.
Instead, in the assignment, you will delve deeper into the BART model that is already trained.
The paper reports various metrics for BART on datasets with all kinds of different types of queries (tail generations). The goal of this project is to investigate whether the model works better for certain types of queries than for others. You will need to think of some experiment that:

- Partitions the evaluation data, and evaluates each partition of the data separately. Note that there are many choices here: you can look at relationships, relationship types, length of queries, length of the data that needs to be generated, etc. Be creative!

- Reflect on the question how you would to compare the results from different partitions: which evaluation metrics will you use, and why?
```
